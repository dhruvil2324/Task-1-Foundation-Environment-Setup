# Task 1 – Foundation & Environment Setup

## Overview

This repository contains the work completed for **Task 1 – Foundation & Environment Setup** of the cybersecurity internship.

The objective of this task was to build a basic cybersecurity lab environment and understand fundamental concepts related to Linux, networking, scanning, packet analysis, and cryptography.

The lab environment was created using **Kali Linux as the attacker machine** and **Metasploitable as the vulnerable target machine** inside a virtualized network.

---

## Lab Environment

**Virtualization Platform**
- Oracle VirtualBox

**Machines Used**
- Kali Linux (Attacker Machine)
- Metasploitable 2 (Target Machine)

**Network Configuration**
- Host-Only Network
- Both machines configured in the same subnet

This setup provides a safe and isolated environment for security testing.

---

## Tools Used

- **Kali Linux** – Penetration testing operating system  
- **Metasploitable 2** – Vulnerable target machine  
- **Nmap** – Network scanning tool  
- **Wireshark** – Network packet analyzer  
- **OpenSSL** – Encryption and cryptographic operations  

---

## Tasks Performed

During this task the following activities were performed:

- Configured Kali Linux and Metasploitable virtual machines
- Verified network connectivity between attacker and target
- Performed port scanning using Nmap
- Captured and analyzed network traffic using Wireshark
- Demonstrated encryption and decryption using OpenSSL
- Practiced essential Linux commands
